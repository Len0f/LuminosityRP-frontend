# LuminosityRP — Frontend

Interface web du projet LuminosityRP, une application destinée à un serveur GTA RP pour consulter et gérer le règlement du serveur. Inclut un panel d'administration permettant d'éditer le contenu sans intervention dans le code.

Site live : [luminosityrp.fr](https://www.luminosityrp.fr)  
Backend : [github.com/Len0f/LuminosityRP-backend](https://github.com/Len0f/LuminosityRP-backend)

## Technologies

- **Framework** : Next.js 12
- **UI** : React 18
- **Requêtes HTTP** : Axios
- **Tests** : Jest, Testing Library
- **Langage** : JavaScript

## Fonctionnalites

- Consultation du règlement du serveur
- Authentification avec gestion des rôles (modérateur, administrateur)
- Panel d'administration pour éditer le règlement sans passer par le code
- Tests unitaires avec Jest et Testing Library

## Structure du projet

- `components/` — Composants React réutilisables
- `context/` — Contextes React (gestion d'état global)
- `pages/` — Pages Next.js (routing automatique)
- `public/` — Fichiers statiques
- `styles/` — Fichiers CSS
- `utils/` — Fonctions utilitaires et appels API
- `jest.config.js` — Configuration des tests
- `next.config.js` — Configuration Next.js
